# Personal Finance Tracker

A comprehensive personal finance tracking application built with React, featuring portfolio management, expense tracking, AI recommendations, and advanced analytics.

## Features

1. **Personal Portfolio** - Track your investments across equities, stocks, mutual funds, and bonds
2. **Expense & Earning Tracker** - Monitor your income and expenses with categorization
3. **Smart Insights & Automation** - Anomaly detection, auto-categorization, and predictive analytics
4. **Goal-Based Finance** - Set micro-goals and track your progress
5. **Advanced Analytics** - Cash flow heatmaps, lifestyle inflation tracking, and needs vs wants analysis
6. **Investments & Net Worth** - Track your net worth over time
7. **Gamification** - Earn streaks, levels, and finance scores

## Installation

```bash
npm install
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)
